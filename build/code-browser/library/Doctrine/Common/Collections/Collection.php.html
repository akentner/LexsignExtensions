  28.  28  * it is essentially an <b>ordered map</b> that can also be used
  29.  29  * like a list.

  31.  31  * A Collection has an internal iterator just like a PHP array. In addition,
  32.  32  * a Collection can be iterated with external iterators, which is preferrable.
  33.  33  * To use an external iterator simply use the foreach language construct to
  34.  34  * iterate over the collection (which calls {@link getIterator()} internally) or
  35.  35  * explicitly retrieve an iterator though {@link getIterator()} which can then be
  36.  36  * used to iterate over the collection.
  37.  37  * You can not rely on the internal iterator of the collection being at a certain
  38.  38  * position unless you explicitly positioned it before. Prefer iteration with
  39.  39  * external iterators.
  40.  40  * 
  41.  41  * @since   2.0
  42.  42  * @author  Guilherme Blanco <guilhermeblanco@hotmail.com>
  43.  43  * @author  Jonathan Wage <jonwage@gmail.com>
